# Lake - The cross-platform 2D Java game engine! 

![showcasedemo.png](readme%2Fshowcasedemo.png)

Lake is a 2D Java game engine under the MIT License, join the fun!

## Features

- Rendering via OpenGL 4.6 and experimental Vulkan 1.3 backends
- Animation API for Spritesheets
- Text Rendering (via Hiero)
- Bezier Curves
- Keyboard/Mouse + Controller Input API
- OpenAL Audio Backend for playing .wav
- Extremely simple 2D physics wrapper around Box2D
- Particle System
- Experimental Immediate-Mode UI system



## Get Started

Download the .jar file and configure it in your classpath to use Lake.
